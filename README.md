# CMU 18-348
This repo serves as base for my own implementation of the CMU's 18-348 Embedded System Engineering course, https://course.ece.cmu.edu/~ece348/. 
I changed the course around somewhat for use on a (somewhat)-modern Nucleo F070RB STM32 microcontroller. 
If snooping around, the most notable achievement is the memory bus (lab 5) implementation.
